# Background
- Upload file from AWS S3 to FTP Server

# How to use
- create `.env` include below information

Example

```
HOST=yourftpserver.com
USERNAME=username
PASSWORD=password
PORT=21
BUCKET_NAME=your_bucket_name
SECRET_ACCESS_KEY=xxxx
ACCESS_KEY_ID=yyyy
REGION=ap-south-east

```
